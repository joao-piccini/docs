## Bem-vindo

Existem diversas aplicações para o uso da API e dos Webhooks, os mais comuns são sincronizar informações dos atendimentos com outros sistemas, você pode conferir todos os endpoints disponíveis abaixo.

O endereço da API é o mesmo endereço da sua plataforma, com a adição da rota `/api/v1` no final da URL, como no exemplo abaixo:

[`https://meusuite.astrow.com.br/api/v1/`](https://meusuite.astrow.com.br/api/v1/)

Após `/api/v1` você pode utilizar os endpoints documentados abaixo, exemplo `/api/v1/atendimento`

Todos os endpoints da documentação possuem um domínio e token genéricos a fins de exemplo, você deve substituir pelo domínio do seu Suite e o token pelo que você gerar.
