## Utilizando tokens

A utilização do token é feita pelo cabeçalho (`Header`) da requisição, conforme demonstrado abaixo:

```
"Authorization": "Bearer $TOKEN"
```

Basta substituir a variável `$TOKEN` pelo seu token gerado na plataforma, conforme demonstrado na seção anterior.
